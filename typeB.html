肥胖類型

醣類型肥胖

肥胖的原因:

身型偏胖中間，以腰腹部最明現(內臟脂肪型肥胖)，皮下脂肪都集中在上半身，大腿屁股反而比較正常。像這種脂肪都堆積在上半身， 尤其隨著年齡的增長，運動量的減少，大小鮪魚肚就默默的，慢慢的爬上身了。 這類型的肥胖，可能是因為吃飯都喜歡吃到10分飽，餐餐都離不開大量的飯或麵。 而且平常喜歡一邊吃飯一邊看電視，或是滑手機，導致常常沒有即時意識到自己已經吃飽了，而繼續把飯吃光光，吃進過多的餐點與熱量。

另外，平常也喜歡吃零食，以及餅乾跟蛋糕，還有愛喝含糖的甜甜飲料。 諸如以上這些飲食習慣，都很容易吃進過多的澱粉、糖類。 尤其是飲料，因為是用喝的，一下子咕嚕咕嚕就一杯下肚，但是一杯珍珠奶茶就含有34克的糖份(136大卡)，熱量超過不說，瞬間吃進這麼多精緻糖，造成血糖的波動，這都是容易造成脂肪堆積的原因。

此外，若內臟脂肪堆積過多， 會影響消化功能，導致內分泌紊亂，還增加了罹患高血脂、高血壓、脂肪肝等疾病的機會...

飲食策略:

對於這一類型的人說，澱粉是熱量過剩的主兇，所以從立定要找回我的腰開始，用餐時要注意

  • 添飯或是點餐時，刻意只添半碗飯，或是讓廚老闆只放一半的麵
  • 餐前先吃點生菜沙拉，或燙青菜，先填滿一半的胃，就可以少吃一點白飯
  • 以五穀飯替代白飯，多嚼一下，吃慢一點，放慢用餐的速度，等控制飽足感的神經中樞開始活動吧！
  • 飯前則借助一些抑制食慾的天然成份例如可可亞萃取跟綠咖啡豆萃取，或是飯前食用洋車前子，喝點促進代謝的茶，都可以立刻增加飽足感，幫助自然降低口慾，減少用餐量。
  • 飯前搭配一些可以阻斷糖份吸收的成份，例如藤黃果、苜蓿、枸杞萃取等等，幫助平緩血糖不要升高太快。

運動策略:

擁有蘋果型身材的你，因為多為均勻胖，所以建議以全身性消脂為出發點，再加入部份局步加強動作。有氧運動可快速消耗熱量，效果最為顯著。

  • 基礎型運動 : 30分鐘的快走很不錯！ 常做上半身左右旋轉的伸展運動，軟化腰間油油圈。
  • 整體性運動 : 從「有氧運動」開始，原地開合跳，跳繩等，關鍵是做間隔性訓練，如2分鐘衝刺跳躍後，踏步兩分鐘，交替重複，會是很好地消耗熱量。
  • 局部塑型運動 : 加入「肌力運動」，以消除大肚子來說，可從各種「捲腹」方是進行全方位擊潰堆積
    • 初階─捲腹運動 :
      1. 仰臥，雙腿彎曲，雙手放於腦後。
      2. 下腰不要離開墊子，胸口向上向前移動至最高點，再緩慢降低至起始姿態。
        (雙手輕輕托頭，不要用力，向上時吐氣，腹肌應有縮短，緊繃的感覺。)
    • 進階─反向捲腹 :
      1. 仰臥，雙腿抬起與地面成90度，雙手放在耳邊，收緊腹部
      2. 肘關節盡量去碰自己的大腿
      3. 脊椎和胸椎要像蛇一樣，一節一節捲起
      4. 如果肘關節碰不到大腿也沒關係，只要腹部肌肉有確實收縮就好。
    • 高階─側身捲腹 :
      1. 身體側臥，左腳彎曲，大腿和小腿的夾角大約成60度，腳板踩到地上
      2. 右腳放在左大腿上，膝蓋同樣彎曲，大腿和小腿的夾角也成60度左右。
      3. 雙手抱在腦後
      4. 上身傾斜卷起，腹腰用力，用左手肘努力去碰右腳膝蓋，然後回復躺平狀態
      5. 每邊做30次，左右交換。
  • 坐著時運動 : 隨時做腹式深呼吸，不僅加強心臟功能，還對消除腹部脂肪、排除腹部廢物，改善腹部血液循環，促進腹部及盆腔臟器的生命活動等，皆有很大的幫助。
